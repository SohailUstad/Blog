What is Java ?

Points covered in this article:
1) Programming kya hai?
2) Computers kaise in programming languages ko samajh pate hain?
3) Programming languages kitne prakar ki hain?
4) Java kya hai?
5) Java kaha use hoti hai ?

Programming kya hai?
Computer ek non-living thing hai. Computer khud se kuchh bhi soch nahi sakta. Computer se koi kaam karaane ke liye hume us kaam ko karne ka poora tarika step by step in detail computer ko samjhana padta hai. Yehi programming hai.

Computers kaise in programming languages ko samajh pate hain?
Lekin computer to humari bhasha nahi samjhta. Computer ko kuchh nahi samjh ata except jo usme pehle se set kiya gaya ho. Computer scientists ne is computer me kuchh chize pehle se hi set kar di hai or unhe hum commands kehte hai. Ab ye kaam scientists ne kaise kiya ispe dhyan dene ki abhi zaroorat nahi hai.

Programming languages kitne prakar ki hain?
Programming languages mainly 3 tarah ki hai. Sabse pehli hai machine language jise hum binary language bhi kehte hai. Isme sirf 2 hi instructions hai(0 and 1). 0 means off and 1 means on. Iske baad ati hai middle level language ise hum assembly bhi keh sakte hai. Iske instructions number ya digit ke form me hote hai. Isi liye to computer ko digital machine kaha jata hai. Har chipset ka instructions set alag hota hai. Akhir me ati hai high level language jo bani hai English ka use krke. Jaise C , C++ , Java , Python etc. Jub bhi ham kisi high level language me koi program likhte hai to wo sabse pehle assembly means middle level language me convert hota hai or phir wo machine language means binary me convert hota hai. Iske baad ye program execution ke liye processor me jata hai or hame desired output milta hai.

Java kya hai?
Java ek high level programming language hai. Ye bahot had tak same c or c++ jaisi hi hai. Hum keh sakte hai ki java c and c++ ka modified version hai. Java ko James Gosling ne 23 January 1996 me launch kiya thha.

Java kaha use hoti hai ?
Java desktop applications, Android applications, web development, big data, embedded systems or game development etc me kaam ati hai.

Java ki kuchh khaas qualities
1- Java platform independent hai yani Java ka jo code hamne windows os ke liye likha hai wohi code bina koi modification or change ke hum Mac os Or Linux os pe bhi run kar sakte hai.
2- Java ek bahot hi simple or sikhne me easy language hai
3- Java secure hai.
4- Java memory management me topper hai.
